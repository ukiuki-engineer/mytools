# 体重管理グラフ

ローカルHTMLで動く体重管理グラフです。
Chart.js CDNを使っているため、サーバーなしで `index.html` をブラウザで開けば使えます。

## 使い方

1. `data.sample.js` をコピーして `data.js` を作成
2. `data.js` に測定値と目標値を入力
3. `index.html` をブラウザで開く

## data.js 形式

```js
const WEIGHT_RECORDS = [
  { date: "2026-05-01", weight: 110 },
  { date: "2026-05-13", weight: 111.3 },
];

const TARGET_RECORDS = [
  { date: "2026-05-01", weight: 110 },
  { date: "2026-06-01", weight: 108 },
];
```

## 方針

- 実測値は点付きの折れ線で表示
- 7日移動平均を線で表示
- 目標体重は入力した日付の点のみを表示し、点同士を折れ線でつなぐ（中間日の補間はしない）
- 表示期間は開始日・終了日を指定して絞り込む
- 7日以上測定が空いた場合でも、移動平均線は直線でつなぐ
- Y軸下限は「表示中データの最小値 - 10kg」を10kg単位で切り下げ
